# GraphQL.js Documentation

GraphQL.js is the official JavaScript implementation of the GraphQL
specification. It provides the parser, validator, executor, type system, and
utilities used to build GraphQL servers, clients, tools, and schema workflows in
JavaScript and TypeScript.
